advice for reffing streamed matches

back to all guides

  • !mp addref the streamer as soon as you make the lobby

  • Post the MP link for the streamer as soon as you make the lobby

  • Don't make players wait for the stream to be ready

  • Before the match starts maybe put on a good song or the TB map so it plays on stream while waiting for the match to start

  • Even during warmups, there must be a player in slot 1 of the lobby for the tourney client

  • Even if a team "instapicks" after a map, give some time before actually changing the map otherwise it can interrupt the previous map finishing or clear the results screen too quickly

  • This is an obvious one, but just keep the ref sheet updated. If a streamer missed something or made a mistake in the overlay, they may use the ref sheet to check picks, bans, and winners.

  • Pay attention for Discord pings because streamers/commentators might be bringing up an issue or asking you something about the match
