This page contains a lot of fundamental information on using OBS Studio and osu!tourney to stream osu! tournaments. It will be broken into many sections for easier navigation, and updated as I write it.

Note that this will not be a step-by-step guide to get you up and running for an osu! tournament. For that, see Nathaniel's guide to streaming, which also contains a detailed guide to using the osu!lazer Tournament Overlay, so I won't cover that here (except for minor notes).

Instead, this is intended as a primer on foundational concepts, vocabulary, and techniques; if you understand this stuff, you won't need a step-by-step tutorial to figure out how to do something.
